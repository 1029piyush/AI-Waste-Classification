Overview

The AI Smart Waste Classification System is a web application that uses deep learning and computer vision to identify waste items from images. The system helps users determine the correct disposal method by classifying waste into categories such as Plastic, Electronic Waste (E-Waste), and Biodegradable Waste. The goal of this project is to support smart waste management and recycling awareness.

Features

Upload an image of waste

AI automatically detects the waste object

Classifies waste into meaningful categories

Displays prediction confidence

Shows whether the waste is recyclable or biodegradable

Provides disposal suggestions

Technologies Used

Python

TensorFlow / Keras

Streamlit

NumPy

Pillow (PIL)

AI Model

The model is trained using a Convolutional Neural Network (CNN) for image classification.
It was trained on a balanced waste classification dataset containing 17 waste categories, including plastic materials, electronic devices, organic waste, and recyclable materials.

Purpose

This system demonstrates how Artificial Intelligence can assist in automated waste sorting and sustainable recycling practices, which are important for improving environmental sustainability and modern waste management systems.
